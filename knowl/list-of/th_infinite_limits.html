Theorem 9.6. Infinite limit formulas.

  1. Let \(n\) be a positive integer. We have
    \begin{align*} \lim_{x\to \infty}x^n \amp= \infty\\ \lim_{x\to -\infty}x^n \amp= \begin{cases} \infty \text{if } n \text{ even}\\ -\infty \text{ if } n \text{ odd} \end{cases}\text{.} \end{align*}
  2. Fix a constants \(a,b\in \R\text{,}\) where \(b> 0\text{.}\) Let \(f\) be a function.
    1. If \(\lim\limits_{x\to a^+}f(x)=0\text{,}\) then
      \begin{equation*} \lim\limits_{x\to a^+}\frac{b}{f(x)}=\begin{cases} \infty \amp \text{if } f(x)> 0 \text{ on open interval } (0,c) \\ -\infty \amp \text{if } f(x)< 0 \text{ on open interval } (0,c) \end{cases} \end{equation*}
    2. If \(\lim\limits_{x\to a^-}f(x)=0\text{,}\) then
      \begin{equation*} \lim\limits_{x\to a^-}\frac{1}{f(x)}=\begin{cases} \infty \amp \text{if } f(x)> 0 \text{ on open interval } (c,0) \\ -\infty \amp \text{if } f(x)< 0 \text{ on open interval } (c,0) \end{cases} \end{equation*}
