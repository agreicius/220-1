Theorem 12.1. Derivative formulas: constant and power functions.

  1. Constant functions.
    For all \(c\in \R\) we have
    \begin{equation} \frac{d}{dx}(c)=0\text{.}\tag{12.1} \end{equation}
    In other words, if \(f\) is the constant function \(f(x)=c\text{,}\) then \(f'\) is the zero function defined as \(f'(x)=0\) for all \(x\in \R\text{.}\)
  2. Identity function.
    We have
    \begin{equation} \frac{d}{dx}(x)=1\text{.}\tag{12.2} \end{equation}
    In other words, if \(f\) is the identity function \(f(x)=x\text{,}\) then \(f'\) is the constant function \(f'(x)=1\) defined as \(f'(x)=1\) for all \(x\in \R\text{.}\)
  3. Power functions.
    Let \(r\) be a real number. We have
    \begin{equation} \frac{d}{dx}(x^r)=rx^{r-1}\text{.}\tag{12.3} \end{equation}
