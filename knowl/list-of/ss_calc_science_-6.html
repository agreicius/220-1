Notation 1.3. Elaborated function notation.

The notation \(f\colon X\rightarrow Y\) simply indicates that \(f\) is a function with domain \(X\) and codomain \(Y\text{;}\) it gives no indication of what the rule is that defines \(f\text{.}\) When we wish to give more information about this rule, we sometimes use the elaborated notation
\begin{align*} f\colon X \amp\rightarrow Y\\ x \amp\mapsto f(x)\text{.} \end{align*}
This is read as “\(f\) is the function with domain \(X\) and codomain \(Y\) that maps an input \(x\) to the output \(f(x)\)”.
