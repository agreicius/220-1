Theorem 23.6.

Assume \(f\) is twice differentiable on the open interval \(I\text{.}\)
  1. If \(f''(x)> 0\) for all \(x\in I\text{,}\) then \(f\) is concave up on \(I\text{.}\)
  2. If \(f''(x)< 0\) for all \(x\in I\text{,}\) then \(f\) is concave down on \(I\text{.}\)
  3. If \(f\) has an inflection point at \(x=c\text{,}\) then \(f''(c)=0\text{.}\)
