Example 20.14. Extreme values: factored polynomial.

Let \(f(x)=(x-2)^5(x-3)^5\text{.}\) Find the absolute maximum and minimum values of \(f\) on the interval \(I=[0,4]\text{.}\)
Solution.
First compute
\begin{equation*} f'(x)=5(x-2)^4(x-3)^5+5(x-2)^5(x-3)^4\text{.} \end{equation*}
Note that \(f\) is differentiable everywhere on the interval \(I\text{,}\) so to find critical points we find all solutions to
\begin{equation*} f'(x)=0 \end{equation*}
lying in \((0,4)\text{.}\) The key technique for solving this equation is factoring:
\begin{align*} f'(x)\amp =0 \\ 5(x-2)^4(x-3)^5+5(x-2)^5(x-3)^4 \amp = 0\\ 5(x-2)^4(x-3)^4((x-2)+(x-3)) \amp =0 \\ 5(x-2)^4(x-3)^4(2x-5) \amp = 0 \text{.} \end{align*}
We now see that the last equality holds if and only if
\begin{equation*} x=2,3,\frac{5}{2}\text{.} \end{equation*}
These are thus our critical points. We now evaluate \(f\) at these critical points, as well as at the endpoints of the interval:
\begin{align*} f(0) \amp = (-2)^5\cdot (-3)^5=2^5\cdot 3^5 \\ f(2) \amp = 0\\ f(5/2) \amp = (1/2)^5(-1/2)^5=-\frac{1}{2^10}\\ f(3) \amp = 0\\ f(4) \amp = 2^5\cdot 1^5=2^5\text{.} \end{align*}
Comparing these values, we conclude that \(f(0)=2^5\cdot 3^5\) is the absolute maximum value of \(f\) on \(I\text{,}\) and \(f(5/2)=-1/2^{10}\) is the absolute minimum value of \(f\) on \(I\text{.}\)
