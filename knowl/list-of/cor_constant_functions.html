Corollary 21.5. Constant function characterization.

Assume \(f\) is differentiable on an interval \(I\text{.}\) If \(f\) satisfies \(f'(x)=0\) for all \(x\in I\text{,}\) then \(f\) is a constant function on \(I\text{.}\) As a consequence, a function is constant on an interval \(I\) if and only if its derivative values are identically equal to zero.
