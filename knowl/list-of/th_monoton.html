Theorem 22.2. Derivative and monotonicity.

Assume \(f\) is continuous on the interval \(I\text{.}\)
  1. If \(f'(a)=0\) for all interior points \(a\in I\text{,}\) then \(f\) is a constant function on \(I\text{.}\)
  2. If \(f'(a)> 0\) for all interior points \(a\in I\text{,}\) then \(f\) is increasing on \(I\text{.}\)
  3. If \(f'(a)< 0\) for all interior points \(a\in I\text{,}\) then \(f\) is decreasing on \(I\text{.}\)
