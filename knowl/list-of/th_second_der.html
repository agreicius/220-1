Theorem 23.10. Second derivative test.

Assume \(f''\) is continuous on an open interval containing \(c\text{.}\)
  1. If \(f'(c)=0\) and \(f''(c)> 0\text{,}\) then \(f(c)\) is a local minimum value of \(f\text{.}\)
  2. If \(f'(c)=0\) and \(f''(c)< 0\text{,}\) then \(f(c)\) is a local maximum value of \(f\text{.}\)
  3. If \(f'(c)=0\) and \(f''(c)=0\text{,}\) then nothing can be concluded about the critical point \(c\text{.}\) In other words, our text is inconclusive in this case.
