Example 20.9. Local and absolute extreme values.

A graph of the function \(f\) on the interval \(I=[-4,4]\) is graphed below.
  1. Identify all and any inputs \(a\) where \(f(a)\) has a local maximum or minimum value of \(f\) on \(I\text{.}\)
  2. Decide whether \(f\) has an absolute maximum value on \(I\text{,}\) and whether \(f\) has an absolute minimum value on \(I\text{.}\)
Solution.
  1. The function \(f\) has a local maximum value at the left endpoint \(a=-4\text{:}\) indeed, we have \(2=f(-4)\geq f(x)\) for all \(x\in [-4,-3)\) (thus for all \(x\) sufficiently close to \(-4\)). The value \(f(4)\) at the right endpoint, somewhat surprisingly, is also a local max: for all \(x\in (3,4]\) we have \(-2=f(4)\geq f(x)\text{.}\)
    Additionally, we see that \(-1=f(-1)\) is a local minimum value of \(f\text{,}\) since \(f(-1)\leq f(x)\) for all \(x\in (-2,0)\text{.}\) Finally, \(f(1)=3\) is easily seen to be the absolute maximum value of \(f\) on \([-4,4]\text{.}\) It follows from RemarkÂ 20.7 that \(f(1)\) is also a local minimum value of \(f\) on \(I\)
  2. We have already observed that \(3=f(1)\) is the absolute maximum value of \(f\) on \(I\text{.}\) We claim that there is no absolute minimum value. To see why, first note that \(f(x)> -4\) for all \(x\in I\text{.}\) Thus, if \(L\) were an absolute minimum value of \(f\) on \(I\) we would have \(L> -4\text{.}\) However, since \(\lim_{x\to 4^+}f(x)=-4\text{,}\) for any \(L> 4\text{,}\) we can find an \(x\in I\) such that
    \begin{equation*} L> f(x) > -4\text{.} \end{equation*}
    It follows that \(f\) has no absolute minimum value on \(I\text{.}\)
